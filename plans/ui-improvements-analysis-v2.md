## 📊 Задача #9: Убрать иконки из модуля редактирования подписки

### Анализ:
✅ **Правильность**: Верно! Иконки занимают много места и не нужны при редактировании.

### Оценка:
- **Сложность**: 🟢 Низкая (5 минут)
- **Влияние**: 🟡 Среднее (UX)
- **Приоритет**: **P2 - Высокий**

### Текущая проблема:
На скриншоте видно, что модуль редактирования показывает выбор иконок, что:
- Занимает много места
- Отвлекает от основной задачи (изменить цену/срок)
- Не нужно при редактировании существующей подписки

### Решение:
```jsx
// SubscriptionModal.jsx
// Скрыть блок выбора иконок при редактировании
{!subscription && (
  <div>
    <label>Icon</label>
    {/* Иконки */}
  </div>
)}
```

### Улучшения:
- Показывать иконки только при создании новой подписки
- При редактировании показывать текущую иконку без возможности изменения
- Или вообще убрать иконки, использовать только логотипы сервисов

## 💡 Детальный план для задачи #9:

### Текущий код (SubscriptionModal.jsx):
```jsx
<div>
  <label>Icon</label>
  <div className="grid grid-cols-4 gap-2">
    {/* 16 иконок */}
  </div>
</div>
```

### Новый код:
```jsx
{/* Показывать иконки только при создании */}
{!subscription && (
  <div>
    <label>{t('modal.icon')}</label>
    <div className="grid grid-cols-4 gap-2">
      {/* Иконки */}
    </div>
  </div>
)}

{/* При редактировании показать текущую иконку */}
{subscription && (
  <div className="flex items-center gap-2 p-3 bg-gray-50 rounded-lg">
    <div className="w-10 h-10 bg-white rounded-lg flex items-center justify-center">
      {getIcon(subscription.iconName)}
    </div>
    <span className="text-sm text-gray-600">
      {subscription.name}
    </span>
  </div>
)}
```
